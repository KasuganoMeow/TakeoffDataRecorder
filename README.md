# Takeoff Data Recorder
Takeoff Data Recorder (ToDR), written in `Rust`, helps you compile your "takeoff" data based on `.fap` files (YAML). 

Please note: that this "takeoff" is not an actual aircraft takeoff and consider the meaning of the word "takeoff" in this context yourself.
